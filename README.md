react-ts-package
======================================

## How to Install
```shell
npm install react-ts-package
```


## How to Run
```typescript jsx
import { Hello } from 'react-ts-package'

const Main = () => {
    return (
        <Hello>
            <p>This is Main Component</p>
        </Hello>
    )
}

export default Main
```
